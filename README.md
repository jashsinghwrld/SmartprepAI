# ExamForge ⚔️

ExamForge is a full-stack AI-driven learning and exam preparation system designed to help students study more efficiently by delivering structured, syllabus-aligned, and exam-focused content.

It integrates intelligent backend processing with a modern frontend interface to provide personalized academic assistance across multiple subjects and modules.

---

## 🔍 Core Features

### 📚 Topic-wise Learning System
Organizes subjects into modules and priority-based topics, enabling structured and strategic preparation.

### 🤖 AI-Powered Q&A Assistant
Uses advanced language models to answer queries, explain concepts, and generate exam-ready responses.

### 📝 Exam Answer Generator
Produces well-structured, marks-oriented answers tailored to university exam patterns.

### 🔎 Smart Search System
Allows users to quickly find topics, concepts, and relevant study material from a curated dataset.

### 📊 Dataset-Driven Insights
Uses pre-built academic datasets to ensure relevance, consistency, and syllabus alignment.

---

## 🛠️ Technology Stack

- **Frontend**: React (Vite), Tailwind CSS, Framer Motion (Animations).
- **Backend**: FastAPI (Python), Procedural Analytics & AI Integration.
- **AI Engine**: Leveraging advanced Large Language Models for dynamic response generation.

## 🚦 How to Run Locally

### Prerequisites
- [Node.js](https://nodejs.org/) (Project built on v18+)
- [Python 3.10+](https://www.python.org/)
- A `GEMINI_API_KEY` in `backend/.env`

### Automatic Launch (Windows)
Run the root-level batch file to install dependencies and start all services:
```powershell
.\start.bat
```

### Manual Launch
**Backend**:
```bash
cd backend
pip install -r requirements.txt
python main.py
```
**Frontend**:
```bash
cd frontend
npm install
npm run dev
```
